# 🌐 Portfolio Website

A responsive personal portfolio website built using **HTML** and **Tailwind CSS** to showcase my skills, projects, and learning journey. This project helped me gain hands-on experience with utility-first CSS, responsive layouts, and modern frontend development.

## 🚀 Features

- 📱 Fully responsive design
- 🎨 Clean and modern user interface
- ⚡ Built with Tailwind CSS
- 🧩 Well-structured and reusable components
- 📂 Dedicated sections for About, Skills, Projects, and Contact
- 🔗 Social media and professional profile links
- 💻 Optimized for desktop and mobile devices

## 🛠️ Tech Stack

- HTML5
- Tailwind CSS
